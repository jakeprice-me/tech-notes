CHMOD Permissions Reference Chart

by David ¬∑ September 18, 2012

This is how I remember permissions and most likely, it will help you remember it as well.

		User	Group	Other
	
	Read	4	4	4

	Write	2	2	2	
	
      Execute	1	1	1

		U	G	O
		X	X	X

Chmods:
777	=	rwxrwxrwx
755	=	rwxr-xr-x
644	=	rw-r--r--
700	=	rwx------
750	=	rwxr-x---
