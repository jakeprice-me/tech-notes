I don't particuarly like extensions on Gnome, and would prefer to use them as little as possible, but the ones below are essential in my opinion.

## Essential Extensions

- [Tiling Assistant - GNOME Shell Extensions](https://extensions.gnome.org/extension/3733/tiling-assistant/)
    - [Leleat/Tiling-Assistant: A GNOME extension which adds a Windows-like snap assist to the GNOME desktop environment. It also expands the 2x2 grid design.](https://github.com/Leleat/Tiling-Assistant)

- [Alphabetical App Grid - GNOME Shell Extensions](https://extensions.gnome.org/extension/4269/alphabetical-app-grid/)
    - [stuarthayhurst/alphabetical-grid-extension: Restore the alphabetical ordering of the app grid, removed in GNOME 3.38](https://github.com/stuarthayhurst/alphabetical-grid-extension)

## Nice to Have

- [Blur my Shell - GNOME Shell Extensions](https://extensions.gnome.org/extension/3193/blur-my-shell/)
    - [aunetx/blur-my-shell: Extension that adds a blur look to different parts of the GNOME Shell, including the top panel, dash and overview](https://github.com/aunetx/blur-my-shell)

- [GSConnect - GNOME Shell Extensions](https://extensions.gnome.org/extension/1319/gsconnect/)
    - [GSConnect/gnome-shell-extension-gsconnect: KDE Connect implementation for GNOME](https://github.com/GSConnect/gnome-shell-extension-gsconnect)

- [Rounded Window Corners - GNOME Shell Extensions](https://extensions.gnome.org/extension/5237/rounded-window-corners/)
    - [yilozt/rounded-window-corners: A gnome-shell extensions that try to add rounded corners for all windows](https://github.com/yilozt/rounded-window-corners)
